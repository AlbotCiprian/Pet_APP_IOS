# Security Policy

## Supported Versions

FlagForge is a scaffold and does not ship releases. Apply updates from the main branch to stay current.

## Reporting a Vulnerability

If you discover a security vulnerability, please open a private issue or email [security@flagforge.io](mailto:security@flagforge.io). Provide as much detail as possible so we can reproduce and address the issue quickly.

We aim to acknowledge reports within 3 business days.
